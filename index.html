What's this?

Karaoke Machine! is a Chrome extension that turns YouTube into your own karaoke machine.

It can karaokize any song by:

  • Separating the vocals and instrumental into individual tracks, so you can mute or unmute either one.
  • Finding the best-matching lyrics online, with line-by-line timing data when available.
  • Extracting precise timing information directly from the song.
  • Displaying synchronized lyrics on top of the video.

Even though it is a Chrome extension, it needs a local backend for the heavy work: downloading audio, separating stems, and timing lyrics.

Install

  1. Download and run the Windows installer.
  2. Open this page in your browser:chrome://extensions
  3. Enable Developer mode and choose Load unpacked.
  4. Select the DKaraoKe folder created by the installer.
  5. Open any YouTube song and click the Press me button to karaokize it!

A CUDA-capable NVIDIA GPU is recommended. CPU mode works, but stem separation and lyric timing can be slow.
